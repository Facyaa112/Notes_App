Веб-приложение для управления заметками с системой авторизации, административной панелью и множеством дополнительных функций.

##  Технологии

### Backend
- **Node.js** - среда выполнения
- **Express.js** - веб-фреймворк
- **JWT (jsonwebtoken)** - аутентификация
- **bcryptjs** - хеширование паролей
- **CORS** - обработка кросс-доменных запросов

### Frontend
- **jQuery** - манипуляции с DOM и AJAX-запросы
- **HTML5/CSS3** - структура и стили
- **Fetch API** - HTTP-запросы

### Инструменты сборки
- **Webpack** - модульный сборщик
- **Babel** - транспиляция JavaScript
- **Nodemon** - автоматическая перезагрузка сервера

##  Функционал

### Основные возможности
-  Регистрация новых пользователей
-  Авторизация с JWT-токенами
-  Создание, просмотр, редактирование и удаление заметок
-  Привязка заметок к конкретному пользователю
-  Пагинация (9 заметок на странице)
-  Поиск по заголовку и содержимому заметок

### Административная панель
-  Просмотр всех зарегистрированных пользователей
-  Статистика по пользователям (количество заметок)
-  Разграничение прав доступа (admin/user)

### Дополнительные функции
-  Редактирование заметок через модальное окно
-  Поиск с debounce (задержка 300мс)
-  Пагинация с навигацией
-  Смена пароля в профиле
-  История действий (Undo/Redo) с горячими клавишами (Ctrl+Z / Ctrl+Y)
-  "Запомнить меня" (сохранение email в cookie)
-  Загрузка изображений по URL
-  Профиль пользователя с редактированием email

##  Установка и запуск

### Требования
- Node.js (версия 14 или выше)
- npm (менеджер пакетов)

### Пошаговая установка

1. **Клонировать репозиторий**

        git clone https://github.com/your-username/notes-app.git
        cd notes-app

2. **Установить зависимости**

        npm install

3. **Собрать клиентскую часть**
        npm run build

4. **Запустить приложение**

        npm run dev

5. **Открыть в браузере**

        http://localhost:3080

### Дополнительные функции
1. **История действий (Undo/Redo)**
    Ctrl+Z - отмена последнего действия
    Ctrl+Y (или Ctrl+Shift+Z) - повтор отмененного действия
    Кнопки на панели управления "Отменить"/"Повторить"
    Хранение последних 10 действий в localStorage

2. **Запомнить меня**
    Автоматическое заполнение email при повторном входе
    Сообщение "С возвращением, [email]!"
    Безопасное хранение (только email, не пароль)

3. **Поиск с debounce**
    Задержка 300мс после ввода
    Поиск по заголовку и содержимому

4. **Пагинация**
    9 заметок на странице
    Навигация с номерами страниц
    Кнопки "Вперед/Назад"
